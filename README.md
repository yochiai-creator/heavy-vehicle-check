# 重機・車両 始業前点検アプリ PWA版

iPhone/Androidで、ホーム画面に追加してアプリみたいに使える版です。

## できること

- フォークリフト、ホイールローダー、ダンプ対応
- 法令ベースの始業前点検項目
- 異常あり時は写真・異常内容・対応内容を必須
- 異常車両は使用不可で保存
- 管理者確認
- CSV出力
- PDF印刷
- QRコード共有
- PWA対応
- ホーム画面に追加対応
- オフライン起動対応

## ローカル確認

```bash
npm install
npm run dev
```

## Vercel公開

1. このZIPを解凍
2. GitHubにアップロード
3. VercelでImport
4. 公開URLを取得
5. アプリ内のQR画面でQRコード作成
6. 従業員はQRを読み込む
7. iPhoneなら Safari → 共有 → ホーム画面に追加

## 注意

写真はファイル名のみ保存する簡易版です。  
写真そのものを全員で共有・長期保存したい場合は、Firebase / Supabase / Google Drive連携が必要です。
